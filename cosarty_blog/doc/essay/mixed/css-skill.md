# css 技巧

## 垂直数字


::: normal-demo 垂直数字

```html
  <div class="demo">
    <span>1</span>
    <span>2</span>
    <span>3</span>
    <span>4</span>
    <span>5</span>
    <span>6</span>
    <span>7</span>
  </div>
```

```css
    .demo{
      writing-mode: vertical-lr;
      text-orientation: upright;
    }
```
:::
